Core Concepts

Screening in Medicine

Key definitions, objectives, and types of screening tests.

What is Screening?

  • Screening aims to presumptively identify unrecognized cases in apparently healthy individuals.
  • Immediate Objective: Exclude healthy people and identify possible cases for further diagnostic proof.
  • Ultimate Objective: Reduce disease-related mortality and morbidity through early intervention.

Test Quality Measures

  • Validity: Capacity of a test to accurately detect the true presence or absence of a condition.
  • Reliability (Repeatability): Test's ability to yield consistent results upon repeated application on the same subject.

Types of Screening

  • Mass Screening: Applied to all individuals regardless of specific risk factors. e.g., Congenital hypothyroidism
  • High-Risk (Selective) Screening: Applied strictly to susceptible groups, yielding more cases at less cost.
  • Opportunistic Screening: Opportunistically testing patients seeking healthcare for other, unrelated reasons.
  • Multiphase Screening: Testing for multiple diseases simultaneously. e.g., Pre-employment profiles

Formulas

Key Formulas

Essential calculations with step-by-step worked examples.

Core Screening Formulas

Sensitivity
Sensitivity = TP / (TP + FN)
Specificity
Specificity = TN / (TN + FP)
Positive Predictive Value (PPV)
PPV = TP / (TP + FP)
Negative Predictive Value (NPV)
NPV = TN / (TN + FN)
Accuracy
Accuracy = (TP + TN) / Total Tested

Worked Examples

Scenario 1: Calculate Sensitivity
A breast cancer screening program evaluates 1000 women. Biopsy confirms 100 women have cancer. Mammogram successfully identifies 50 of these women.
Show Steps
1 Identify True Positives (TP) = 50
2 Identify total diseased = 100
3 Sensitivity = 50 / 100
Sensitivity = 50%
Scenario 2: Calculate Specificity
From the same 1000 women, 900 do not have cancer. The mammogram correctly returns a negative result for 750 of these women.
Show Steps
1 Identify True Negatives (TN) = 750
2 Identify total healthy = 900
3 Specificity = 750 / 900
Specificity = 83.3%
Scenario 3: Calculate PPV
A total of 200 women test positive on the screening mammogram, but only 50 actually have biopsy-confirmed cancer.
Show Steps
1 Identify True Positives (TP) = 50
2 Identify total positive tests = 200
3 PPV = 50 / 200
PPV = 25%
Scenario 4: Calculate Accuracy
Using the previous data: 50 true positives, 750 true negatives, out of 1000 total women.
Show Steps
1 TP + TN = 50 + 750 = 800
2 Total population = 1000
3 Accuracy = 800 / 1000
Accuracy = 80%
Exam Traps

Common Exam Traps

High-yield pitfalls to avoid on test day.

Validity ≠ Reliability Confusion

A valid test is mathematically always reliable, but a reliable test is not inherently valid. Do not confuse the two — reliability means consistency, not accuracy.

Screening ≠ Diagnosis Action

Never initiate treatment based solely on a screening test; it must be confirmed by a diagnostic test. Screening only identifies possible cases.

Target Audience Error

Do not apply screening test algorithms to symptomatic patients. Screening is exclusively for apparently healthy individuals.

False Negative Danger

Highly fatal diseases demand high sensitivity because incorrectly reassuring a sick patient (false negative) has disastrous consequences.

False Positive Danger

Olympic doping tests prioritize high specificity to prevent falsely accusing clean athletes. Think: specificity protects the innocent.

Screening Cost-Efficiency

Mass screening is often less cost-effective; high-risk (selective) screening identifies more cases for less money. Don't confuse broader with better.

Disease Prerequisites

A disease must have a recognized latent or asymptomatic period to be suitable for screening. Without this window, screening cannot occur before clinical presentation.
